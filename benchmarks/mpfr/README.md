# MPFR Benchmarks

This directory is reserved for MPFR/MPC benchmark families.

The GMP/MPF benchmark families live under `benchmarks/gmp/`.  Future MPFR/MPC
benchmarks should mirror that layout:

- `00_Rdot/`
- `01_Raxpy/`
- `02_Rgemv/`
- `03_Rgemm/`
